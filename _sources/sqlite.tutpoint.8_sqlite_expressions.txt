##################
SQLite Expressions
##################
**http://www.tutorialspoint.com/sqlite/sqlite_expressions.htm** 

- `Expressions (MSDN) <https://msdn.microsoft.com/en-us/library/ms190286.aspx>`__ -- a combination of symbols and operators that the SQL Server Database Engine evaluates to obtain a single data value. Simple expressions can be a single constant, variable, column, or scalar function. Operators can be used to join two or more simple expressions into a complex expression.
